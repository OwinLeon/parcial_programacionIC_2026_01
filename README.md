# Punto 1 – Condicionales
## Evaluación del rendimiento de un empleado

Una empresa desea clasificar el desempeño mensual de sus empleados a partir de una puntuación obtenida en evaluación interna.

El programa debe solicitar:
- Nombre del empleado
- Documento o identificación
- Puntaje de desempeño (de 0 a 100)

Según el puntaje, el sistema debe clasificar al empleado así:

- De **0 a 59**: desempeño insuficiente
- De **60 a 74**: desempeño aceptable
- De **75 a 89**: buen desempeño
- De **90 a 100**: desempeño sobresaliente

### El programa debe mostrar:
- Nombre del empleado
- Documento
- Puntaje obtenido
- Clasificación correspondiente

### Condiciones adicionales
- Si el puntaje ingresado está fuera del rango de 0 a 100, debe informarse que el valor es inválido.
- El mensaje final debe ser claro y bien organizado.

### Lo que se espera evaluar en este punto
- Uso correcto de condicionales
- Validación de rangos
- Salida organizada por consola
- Lógica adecuada en la clasificación

# Punto 2 – Ciclos con arreglos lineales
## Control de calificaciones de un grupo

Un docente necesita registrar las calificaciones de un grupo de estudiantes para obtener algunos indicadores rápidos del curso.

Construya un programa que permita:

1. Solicitar la cantidad de estudiantes que se van a registrar.
2. Crear un arreglo lineal con ese tamaño.
3. Ingresar la nota de cada estudiante.
4. Recorrer el arreglo y calcular:
   - La nota más alta
   - La nota más baja
   - El promedio del grupo
   - Cuántos estudiantes aprobaron
   - Cuántos estudiantes reprobaron

Considere que un estudiante **aprueba con nota mayor o igual a 3.0**.

5. Mostrar todas las notas ingresadas.
6. Mostrar un resumen general del grupo.

### Consideraciones
- Debe usar un arreglo lineal para almacenar las notas.
- La lectura de datos debe hacerse con ciclos.
- El procesamiento del arreglo también debe realizarse con ciclos.
- La solución debe adaptarse a la cantidad de estudiantes digitada por el usuario.

### Lo que se espera evaluar en este punto
- Arreglos lineales
- Ciclos de entrada y salida
- Recorrido de arreglos
- Cálculo de promedio
- Uso de contadores y comparaciones
